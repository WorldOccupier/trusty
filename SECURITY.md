# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Trusty, please report it privately by emailing the maintainers at the email address in the GitHub profile.

Please do NOT file public issues for security vulnerabilities.

## Supported Versions

| Version | Supported |
|---------|-----------|
| >= 0.1.0 | Yes |

## Responsible Disclosure

We kindly request that you give us reasonable time to address the issue before disclosing it publicly.
